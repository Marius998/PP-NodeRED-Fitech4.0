# PP-NodeRED-Fitech4.0

## Node-RED Setup 
### Installation 
`sudo npm install -g --unsafe-perm node-red`

Docker `docker run -it -p 1880:1880 --name mynodered nodered/node-red`

Snap `sudo snap install node-red`

### Import flows

Start `node-red`

Import **PP-NodeRED-Fitech4.0/flows.js**
